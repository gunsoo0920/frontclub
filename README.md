> 단순한 도서 검색을 넘어, 감상평을 공유하고 소통하는 **SPA 기반 소셜 리딩 플랫폼**입니다.

## 1. 프로젝트 개요 (Project Overview)

- **프로젝트명:** 서평 클럽 (Owl Club)
- **개발 기간:** 2025.10.13 ~ 2025.12.15
- **팀명:** 올빼미클럽 (4인)
- **목표:** React의 **컴포넌트 구조**와 **Hooks(`useState`, `useEffect`)**를 활용하여 데이터 흐름을 제어하고, **사용자 중심(UX)**의 웹 서비스를 구축합니다.

<br>

## 2. 실행 방법 (How to Run)

교수님, 본 프로젝트는 **React**와 **JSON Server**를 사용하여 개발되었습니다.
원활한 실행을 위해 **터미널 2개**를 열어 각각 실행해 주시길 부탁드립니다.

### 📥 1) 패키지 설치

```bash
npm install
```

### 🗄️ 2) Backend 실행 (JSON Server)

- 데이터베이스 역할을 하는 가상 서버를 4000번 포트(또는 설정된 포트)로 실행합니다.

```bash
# data.json이 있는 경로에서 실행해주세요.
npx json-server ./data.json --port 4000
```

### 💻 3) Frontend 실행 (React App)

- 웹 애플리케이션을 실행합니다.

```bash
npm start
```

## 3. 주요 기능 및 기술적 특징 (Key Features)

### 🔐 1. 회원 서비스 & 보안 (Auth & Security)

- **금융권 수준의 세션 관리:** `localStorage`와 이벤트 리스너를 연동하여 **5분 미활동 시 자동 로그아웃** 기능을 구현했습니다.
- **활동 감지:** 마우스 이동이나 클릭 시 세션 시간이 자동으로 연장됩니다.
- **유효성 검사:** 회원가입 시 **Regex(정규식)**을 사용하여 이메일과 전화번호 형식을 엄격하게 검증합니다.
- **계정 찾기:** 이름과 전화번호 매칭을 통한 아이디/비밀번호 찾기 기능을 제공합니다.

### 📚 2. 도서 목록 시스템 (List & Sorting) - ✨ Updated!

- **다중 정렬 기능:** `sort` 함수를 활용하여 **[리뷰 많은 순 / 가나다 순]**으로 리스트를 즉시 재정렬할 수 있습니다.
- **카테고리 필터링:** 5개 대분류(IT, 에세이 등) 탭을 통해 데이터를 실시간으로 필터링합니다.
- **검색 최적화:** 무분별한 요청 방지를 위해 **2글자 미만 검색 제한(Validation)**을 적용했으며, 뒤로 가기 시에도 검색 결과가 유지됩니다.

### 📝 3. 상세 & 리뷰 관리 (CRUD)

- **권한 분기 처리(Authorization):** 로그인 상태와 작성자 본인 여부를 체크하여 **[수정/삭제] 버튼을 조건부 렌더링**합니다.
- **데이터 방어:** 비로그인 유저가 URL로 강제 접근 시 로그인 페이지로 **리다이렉트(Redirect)** 처리됩니다.
- **비동기 데이터 처리:** REST API(`fetch`)를 활용하여 리뷰 데이터를 실시간으로 Create, Read, Update, Delete 합니다.

### 🎨 4. UX & UI Animation

- **스크롤 리빌(Scroll Reveal):** `Intersection Observer API`를 커스텀 훅으로 구현하여, 스크롤 시 컨텐츠가 부드럽게 떠오르는 애니메이션을 적용했습니다.
- **스크롤 위치 제어:** 페이지 이동 시 스크롤이 최상단으로 초기화되도록 `ScrollToTop` 컴포넌트를 적용했습니다.
